# Тестовое задание  

Проект автотестов для сервиса Яндекс.Диска https://yandex.ru/dev/disk/rest/

## Структура проекта


    ├── src
    │   └── test/
    │        ├── java/
    │        │    ├── tests/                            - Папка с тестами
    │        │    │    ├── BaseTest                     - Базовый класс для тестов
    │        │    │    ├── CopyFileTest                 - Тесты на копирование файла
    │        │    │    ├── CreateFileTest               - Тесты на создание файла
    │        │    │    ├── CreateFolderTest             - Тесты на создание папки
    │        │    │    ├── CreateFolderTest             - Тесты на удаление папки
    │        │    │    ├── DownloadFileWithContentTest  - Тест на скачивание файла с контентом
    │        │    │    └── RecoveryFolderTest           - Тесты на восстановление папки
    │        │    └── utils/                            - Папка для вспомогательных классов
    │        │         ├── FileFactory                  - Класс для создания файла
    │        │         └── PropertyProvider             - Провайдер конфигурационных свойств
    │        └── resources/
    │             ├── config.properties                 - Файл конфигурации
    │             └── config.properties.template        - Шаблон файла конфигурации
    ├── gitignore                                       - Список файлов и папок, которые Git должен игнорировать  
    ├── pom.xml                                         - Зависимости проекта
    ├── README.md                                       - Описание проекта и инструкция
    └── TESTCASES.md                                    - Тест-кейсы

## Инструкция

1. Склонируйте к себе репозиторий, в котором хранится проект тестового задания. В командной строке/терминале выполните команду

       git clone https://github.com/EldarAbdulhakov/YDisk

2. В папке resources скопируйте config.properties.template в config.properties  
   (В папке resources создайте файл config.properties, скопируйте в него данные из config.properties.template)
3. Впишите свой токен Яндекс API в config.properties: yandex.auth.token=<ваш токен Яндекс API>  
Получение токена:  

       1. Авторизоваться в Яндекс ID
       2. Перейти в Полигон по ссылке: https://yandex.ru/dev/disk/poligon/
       3. Получить токен, нажав на кнопку “Получить OAuth-токен”
4. Убедитесь, что на вашем компьютере установлена Java 21. В командной строке/терминале выполните команду

       java -version

   Если команда не распознаётся — убедитесь, что Java добавлена в переменную среды PATH.
5. Убедитесь, что на вашем компьютере установлен Maven. В командной строке/терминале выполните команду

       mvn -version

   Если команда не распознаётся — убедитесь, что Maven добавлен в переменную среды PATH.

6. Запустите тесты. В командной строке/терминале из папки проекта выполните команду

       mvn clean test
