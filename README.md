# Тестовое задание

## Структура проекта

    ├── tests/                    - Папка с тестами
    │   ├── AsdListMobileTest     - Тесты страницы с объявлениями на мобильной версии сайта
    │   ├── AdsListTest           - Тесты cтраницы с объявлениями
    │   ├── BaseTest              - Базовый класс для тестов
    │   ├── MobileBaseTest        - Базовый класс для тестов мобильной версии сайта
    │   └── StatsTest             - Тесты страницы статистики
    ├── utils/                    - Папка для вспомогательных классов
    │   └── WaiterUtils           - Класс с методами-ожиданиями
    │   └── 
    ├── pom.xml                   - Зависимости проекта
    ├── README.md                 - Описание проекта и инструкция
    └── TESTCASES.md              - Тесткейсы

## Инструкция

1. Склонируйте к себе репозиторий, в котором хранится проект тестового задания. В командной строке/терминале выполните команду

       git clone https://github.com/EldarAbdulhakov/qa-internship-task

2. Убедитесь, что на вашем компьютере установлена Java 21. В командной строке/терминале выполните команду

       java -version

   Если команда не распознаётся — убедитесь, что Java добавлена в переменную среды PATH.

3. Убедитесь, что на вашем компьютере установлен Maven. В командной строке/терминале выполните команду

       mvn -version

   Если команда не распознаётся — убедитесь, что Maven добавлен в переменную среды PATH.

4. Запустите тесты. В командной строке/терминале из папки проекта выполните команду

       mvn clean test
